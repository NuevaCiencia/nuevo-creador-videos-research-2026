# Fonts

Esta carpeta almacena las tipografías instaladas para su uso en la aplicación.

A diferencia del resto de las carpetas dentro de `/data/`, el contenido de `/data/fonts/` **SÍ se sube y controla con el repositorio Git**, ya que las tipografías son dependencias del proyecto necesarias para mantener el diseño visual idéntico al renderizar en otras máquinas.

Coloca aquí cualquier archivo `.ttf` o `.otf` que necesites y nómbralo en la configuración del proyecto para que FFmpeg/Remotion pueda usarlo.
