# Exports

Esta carpeta almacena de forma automática una copia de seguridad (backup) en formato `.json` cada vez que descargas el estado de una clase usando el botón de exportación en la interfaz de usuario.

El formato del nombre del archivo es:
`{nombre_del_proyecto}_{nombre_de_la_clase}_{fecha}.json`

Esta carpeta es ignorada por el repositorio Git (excepto este archivo `notas.md`) para evitar subir los respaldos locales al control de versiones.
