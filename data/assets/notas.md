# Assets

Esta carpeta almacena todos los insumos multimedia generados por el sistema o subidos por el usuario, organizados por el ID de la clase.

- `/assets/{class_id}/images/` -> Imágenes generadas, subidas o segmentadas.
- `/assets/{class_id}/videos/` -> Videos generados, subidos o animaciones de Remotion.
- `/assets/{class_id}/audio/` -> Archivos de audio originales y la transcripción base.

Esta carpeta es ignorada por el repositorio Git (excepto este archivo `notas.md`) para evitar subir contenido autogenerado.
