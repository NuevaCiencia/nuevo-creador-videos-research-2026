# Database

Esta carpeta almacena la base de datos principal de la aplicación.

- `video_creator.db` -> La base de datos SQLite con los proyectos, clases, guiones y estado.
- `video_creator.db.bak` -> Un respaldo (backup) automático generado cada vez que se reinicia el servidor.

Esta carpeta es ignorada por el repositorio Git (excepto este archivo `notas.md`) para asegurar que la información personal y los avances de los proyectos no se filtren ni generen conflictos de versiones en Git.
